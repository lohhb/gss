Subject: SCS NS8000 evaluation HOT Jam for 4x4.5 encounter device pick up problem

Handler Model: NS8080 (S/N: 181417, SCS), HMI Ver 2.0

Controller: RC520

Date: 3 Mar 2008


 

Symptom

Device float/tilt on tray

Handler should be able to overcome this problem but due to different height  of Input pick up hand, unable to achieve a pickup height to overcome this problem. MCAL, 4 hands not same height.


 

Action

Auto Height Teaching but Hand A and Hand C different height. Edited HOFS manually in SPEL CT (selrb2) to achieve 4 hands same pick up height.
