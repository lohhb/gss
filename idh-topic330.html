Subject: Input & Output Arm Y-Axis  F5041 Torque error in Low Power State

Handler Model: NS-6040 (UTAC, NS60-15T)

Controller: SRC520CE

Date: 3 Dec 2007


 

Symptom

NS60-15T encounter Error 5041 Torque error in low power state on Y-Axis motors when executing MCAL for Input or Output Arm.


 

Action

Check Y Axis timing belt tension, ok

Check connections to Y Axis motors.

Dummy Inertial weight missing from both pulley of Y-Axis Input & Output Arm.


 

Cause
