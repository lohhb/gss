Technical Support Notes

Subject: NS-6040 CPU Board Upgrade PC-686 BX (Windows 2000 / SPEL CT)

Date: 18 Dec 2007

Backup

1) Setup files (By Export in MMI)

2) [Config] folder (in C:\NS6000)

3) [Run] folder (in C:\NS6000)

4) SPEL Project in SPEL95

5) SPEL SYSTEM in SPEL95

6) DU 1, 2, 3 data in SPEL95

7) Robot Manipulator in SPEL95


 

Hardware Replacement

8) CPU Board Replacement (For Type 2 & 3, need to change CPU Fan with thermal paste and it's harness)

9) * Replace MIB boards for Type 1 & 2

10) * Replacement to System Panel II for Type 1

11) * Replacement to 2.5” HDD for Type 1

12) Remove sub panel board(RS232 port with parallel port)

13) Remove LAN card

14) Insert new Parallel Port

15) Insert new COMM Port (COMM 1 & 2)


 

Recovery

16) in BIOS, set  [PnP/PCI Configurations] to Yes, Resources controlled By [Manual], Set IRQ 5 & IRQ 10 to Legacy ISA

17) Use Recovery CD (FD07S057, ver 3.0c, 2007/06/20) to do recovery of Windows 2000, MM1 and SPEL CT(CPU 850MHz)

18) Install PC I/O driver (Multimedia controller) (Contec Driver in Recovery CD)

19) Install IFUN100 driver (Driver in Recovery CD)

20) Use Loader.exe (in C:\NS6000\Exe) to load hex file (Hex file in Recovery CD)

21) Restore DU 1, 2 & 3 data in SPEL CT

22) Restore Robot Manipulator data in SPEL CT

23) Copy pos data files from backup project to SPEL CT project

24) Copy & paste backup [RUN] folder to C:\NS6000

25) Copy & paste backup [Config] folder to C:\NS6000

26) Edit “Product.dat” in C:\NS6000\Log folder to change the Handler Serial Number
