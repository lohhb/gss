Technical Support Notes

Subject:  CPU upgrade “Power down” in MMI status after restore SPEL System

Handler Model: NS-6040 (S/N: 12731), Type 3

Customer No: H20-NS64

Controller: SRC-520 S/N: 02617

Date: 11 Dec 2007


 

Symptom

Performed CPU Upgrade to PC-686BX(PC)-EPC850 on H20-NS64 (S/N: 12731), Type 3.

Encounter MMI status windows show “Power down” after restore SPEL System, suspect incompatible SPEL SYSTEM data.

Pentium 200MHz CPU Board (Spec. CU-M313) to PC-686BX(PC)-EPC850

Win95 -> Win2000

MMI 2.1b->MMI 3.0c

SPEL95 ver 2.53i -> SPEL CT 3.0c


 

Action

Use Recovery CD (FD07S057, ver 3.0c, 2007/06/20) to do recovery again.

Restore DU 1,2 & 3 data in SPEL CT.

Restore Robot Manipulator data in SPEL CT.

Copy & paste old pose data to SPEL CT project.

Handler able to dry run and dummy device alignment.


 

Cause

SPEL CT is only supported by Windows 2000. The SPEL95 SYSTEM file is not compatible with SPEL CT.


 

Remarks

Refer procedure on NS-6040 CPU Board Upgrade PC-686 BX (Windows 2000 / SPEL CT)
