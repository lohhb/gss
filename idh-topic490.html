Technical Support Notes

Subject: NS-7000 MMI downgrade version downgrade issue

Handler Model: NS-7000

Controller: RC520

Date:


 

Symptom

some minor problem during MMI downgrade from v2.0a to v1.1c13.  Firstly backup system data, pose data and setup files. Perform HDD recovery using CD-ROM Ver.1.1 (FD05S50), after that restore system data, pose data and setup files and then version up using Ver.1.1c13 CD. The error " File reading error c:\NS7000\Config\Table\DIO_If\27.dat" occurred when DeviceSet/Tester IF icon is select.


 

Action

The normal situation of Ver.1.1c13, DIO setting files does not have 27.dat.

For this case, the 27.dat file is not present in C:\NS7000\Config\Table\DIO_If folder.

*There is a possibility that customer registered original DIO setting as the 27.dat.

Please ask to customer for original DIO setting of 27.dat and please register again original DIO setting as 27.dat. (Refer MMI operation manual 6.6 DIO Interface Configuration)
