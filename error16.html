Error 16   Statement not allowed outside of function body.

Causes

The program has statements after the last FEND statement.

Program line is not within FUNCTION - FEND.

Remedies

Move all statements past the FEND statement to before the FEND statement.

Include program lines between FUNCTION - FEND.
