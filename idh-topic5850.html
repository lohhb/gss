Service History

Subject: NS-8040 Index Arm 1 crashed (caused nests cracked)

Handler Model: NS-8040 ( S/N: 181213)

Controller: RC520

Date: 24 Mar 2008


 

Symptom

Eval NS8040(S/N: 181213) Index Arm 1 crashed on 24 Mar 08 night due to contactors hooked in the wrong orientation.

2 nest cracked. Suspected contactor heads for Arm 1 fitted with wrong orientation. Socket Guide Base's(plate) guide pins penetrate through nest and pick up tip cracked.


 


 

Action

Verified coplanarity with respect to socket guide plate, no visible slant of Arm 1.


 

Reteach Index Arms P50, P51, Shuttle 1's P21 & P22. Shuttle 2's points ok.

Install HMI Ver C 8.10-SIK (FD08S0025) on 10% speed  for Index Arms feature after Pause, tested ok.

Alignment check/adjust for shuttles and Auto1,2,3. AutoContact height for Index. Dock with tester and run production. Production monitoring.

May need Ijima-san to drop by as rubbing of nest and socket guide base pin can be heard during plunging into test site of Arm 1.


 

Cause

Contactors hooked in the wrong orientation for Index Arm. Test-site X pitch 60mm for LF2BGA189-1 change kit. When Contactors hooked in wrong orientation, the X-pitch between 2 contactors becomes 100mm, so it crashed on the socket guide plate that has X pitch for 2 pocket 60mm.
