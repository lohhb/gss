Technical Support Notes

Subject: NS-7000 Shuttles vibration

Handler Model: NS-7000 (Amkor)

Controller: RC520

Date: 15,17 & 23 Jan 2008


 

Symptom

Shuttle 2 vibration lead to index arm 2 unable to pick up device.


 

Action

15 Jan 2008

Checked CT data. OK

Shuttle 1 and 2 swap servo pack(motor driver module). Same

Updated manipulator settings with reducer for shuttle motors. Same

Used mcd file from another “ok” handler. Same

Tightened belt tension. Situation improved. Intermittent vibration.

Shuttle pick up height calibration and increased pickup 0.2mm.

Contact height calibration increased 0.45mm for Bin 1.

Suzuki-san to check if possible 17/Jan. Intermittent vibration.


 

17 Jan 2008

Handler still has Index Arm 2 pick up error based on error log. Verified P20 and P21 with 0.5mm difference. Motor reducer found vibrating intermittent, Due to tight production, only managed to verify SPEL point.


 

23 Jan 2008

Received new mcd files from Japan. Due to tight production, tired on offline handler before trying on production handler.

Tried new mcd file, found N.G. Same problem

Extract mcd files from handler with no vibration for comparison

Edited with Mcdsrv.exe accordingly. Try different values till no vibration. Total 3 sets for comparison.640->500 and 440->250

Edited with Mcdedit.exe to remove LP error

Verified few trays running, no vibration.

Upon confirmation this mcd is ok, patch on 2 handlers and no vibration.


 

24 Jan 2008

Device pick up error and drop error on Index Arm 2. Intermittent failure on both sites.

Contact height found 0.9mm deeper.

Undock and found screw stuck in cylinder for dip board lock

Dip board check and Socket change

After contact height, increase 0.8mm to achieve stable yield with 0.2s delay

Found index arm and SLK tubing loose.

Vacuum sensitivity on arm 2 head 2 delay. Tunning done

Monitored few trays production. Yield stable and no jam.
